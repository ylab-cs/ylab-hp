# 山口研究室HP(VuePress)

![logo](./docs/.vuepress/public/assets/img/logo.png)

## 前提環境

- node.js v10 以上
- npm 6.9.0 以上

## 環境構築

必要パッケージをインストール

```bash
npm install
```

## 開発サーバ起動

```bash
npm run docs:dev
```

http://localhost:8080 で起動するのでブラウザで開く

## ページの編集

/docs 内の .md ファイルを編集する

## その他カスタマイズする場合

公式ドキュメント参照

https://vuepress.vuejs.org/
